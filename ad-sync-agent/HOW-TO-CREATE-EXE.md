# Как создать готовый .exe файл для распространения

## Для разработчика/администратора (один раз)

Чтобы создать готовый standalone .exe файл для распространения:

### Windows:

1. **Установите Python 3.8+**
   - Скачайте с https://www.python.org/downloads/
   - При установке отметьте "Add Python to PATH"

2. **Распакуйте архив** в любую папку

3. **Запустите `build-standalone-exe.bat`**
   - Скрипт автоматически установит зависимости
   - Создаст готовый .exe файл в папке `dist\`
   - Это займёт 5-10 минут

4. **Готово!** Файл `dist\AD-Sync-Agent.exe` готов для распространения

### Что получается:

- ✅ **Standalone .exe** файл (~50-100 MB)
- ✅ **Не требует Python** - всё включено
- ✅ **Не требует зависимостей** - всё встроено
- ✅ **Просто запустить** - двойной клик

## Для конечного пользователя

1. **Скачайте** `AD-Sync-Agent.exe`
2. **Скачайте** `ad_sync_agent.py` (в ту же папку)
3. **Запустите** `AD-Sync-Agent.exe`
4. **Заполните** настройки в GUI
5. **Запустите** агента!

**Всё! Никаких установок не требуется!**

## Рекомендации

- Создайте .exe файл один раз при выпуске новой версии
- Включите готовый .exe в архив для скачивания
- Пользователи просто скачивают .exe и запускают
